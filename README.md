## 스터디 규칙 

### ✅ 진행 방식
- 주 2회 온라인(WebEx) 미팅
- 매주 월요일 풀고 온 문제 풀이 및 코드 리뷰
- 매주 목요일 모의 코딩테스트 진행, 코드 리뷰


### ✅ commit 규칙
- commit 메세지: [플랫폼] 번호_문제이름
- description: 문제 주소 (option)
- 터미널에서 작성법: 
```
git commit -m "[BOJ] 21554_마법사의 돌 장난감"
```
- 플랫폼 작성법 통일: 
  * [BOJ] - 백준 
  * [SWEA] - 삼성SW Expert Academy
  * [PGS] - 프로그래머스
  * [ETC] - 그외
